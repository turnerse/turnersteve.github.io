# turnersteve.github.io

Personal website for Steve Turner, hosted at [turnersteve.com](https://turnersteve.com) via GitHub Pages.

## Stack

Plain HTML and CSS — no build step, no framework, no package manager.

- `index.html` — all page content
- `styles.css` — all styles (CSS custom properties, glassmorphism card layout)
- `assets/` — local images
- `CNAME` — custom domain config

## Deployment

Push to `main` → deploys automatically via GitHub Pages.
